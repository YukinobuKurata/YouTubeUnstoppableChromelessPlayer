YouTubeUnstoppableChromelessPlayer
==================================

「YouTube動画の一時停止ができないようにする仕組み （Chromeless Player（クロムレスプレーヤー））」

&lt;iframe&gt;タグの埋め込みコードをつかったHTML5形式の動画プレーヤーです。
